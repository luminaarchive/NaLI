# Templat Pengumpulan Umpan Balik Penguji (Tester Feedback Collection Template)

Dokumen ini berisi templat salin-tempel (copy-paste) yang dapat digunakan oleh Founder untuk mendokumentasikan hasil pengujian dari masing-masing penguji (3–5 orang) secara terstruktur.

---

## 1. Templat Salin-Tempel Umpan Balik (Copy-Paste Template)

```text
Tester name: 
Device: 
Browser: 
Date: 
Prompt tested: 
Result summary: 

Was output useful? 1–5: 
Was evidence warning clear? 1–5: 
Was mobile usable? 1–5: 

Did anything break? 
Did anything feel misleading? 
Would you use it again? 
Would you pay later? 

Screenshot/video link if any: 
Top issue: 
Top improvement: 
```

---

## 2. Interpretasi Skor Penilaian (Scoring Interpretation Rubric)

Gunakan panduan skor berikut saat mengisi kolom bernilai 1-5 di atas:

| Skor | Label Kualitas | Interpretasi UX / Fungsionalitas |
| :---: | :--- | :--- |
| **5** | **Excellent** | Sempurna. Tidak ada kendala visual/fungsional, alur sangat intuitif, hasil sangat sesuai ekspektasi. |
| **4** | **Usable** | Sangat layak pakai. Berfungsi dengan baik, ada sedikit saran kosmetik/copywriting minor tetapi tidak mengganggu alur utama. |
| **3** | **Acceptable but needs improvement** | Cukup/bisa diterima. Aplikasi dapat menyelesaikan tugas, tetapi ada kebingungan minor pada navigasi atau teks instruksi. |
| **2** | **Friction** | Mengalami hambatan. Ada masalah layout tumpang tindih di perangkat tertentu, atau respons lambat/kurang memuaskan yang memerlukan perhatian segera. |
| **1** | **Blocker** | Masalah kritis/gagal total. Halaman crash (blank screen), tombol penting tidak merespons, atau model AI menghasilkan informasi ilegal/tidak aman. |
