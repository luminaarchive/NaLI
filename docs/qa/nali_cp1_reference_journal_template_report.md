# Reference Journal Template Verification Report

This report documents the verification of the NaLI-branded academic journal template, which adapts the structure and presentation discipline of the benchmark journal format.

## Reference Structure emulated:
1.  **Journal Cover**: Large bold typography, issue & volume markers, publisher details, and natural/wildlife thematic graphic placeholder.
2.  **First Article Page**: Running banner, double horizontal rules, clear author & affiliation formatting, shaded Abstract block with narrowed margins, keywords section, and a structured Article Information block (milestones).
3.  **Body Pages**: Dense structured margins, running header citation, standard headers (PENDAHULUAN, TINJAUAN PUSTAKA, METODOLOGI, HASIL DAN PEMBAHASAN, KESIMPULAN, DAFTAR PUSTAKA).
4.  **Closing Elements**: Formatted tables for results, visual/photo evidence slot boxes, limitations warn callout, Annexure, and References lists.

---

## 1. What was structurally copied
- The layout metadata layout (Volume, Issue, Year, ISSN, DOI placement).
- The narrower margin spacing, shaded background, and header details for the Abstract section.
- The use of clear section divisions and horizontal ruling.
- Comparative Results Table structure (Spesimen comparison).
- Running headers and footers with page number structures.

## 2. What was NOT copied (NaLI Brand Protection)
- **No Publishers Identity Copied**: The E-Palli / JWC branding, logos, copyright markings, and publisher addresses were excluded. The template uses **NaLI Nature & Evidence Journal** and **NatIve Field Intelligence Services** branding.
- **No Fake Identifiers**: Real ISSNs and DOIs were not fabricated. The template uses:
  *   `DOI: Not assigned in CP1 (Draft stage)`
  *   `ISSN: Not applicable / internal QA draft`
- **No Plagiarism Evasion**: No fake reference citations or literature gaps were invented. If no references are provided by the user, the References section states: *"No references were supplied. NaLI did not generate artificial references."*
- **No Upload Claim**: Visual photo cards are explicitly marked: *"Foto belum disediakan (Upload file tidak aktif di CP1)"*.

## 3. Model Adaptations

*   **Peregrine**: A concise, starter layout with simple phrasing and 3 keywords.
*   **Obsidian**: A highly rigorous evidence limitation draft highlighting n=2 sample count, lack of photo validation, ketiadaan instrumen, and determinasi taksonomi.
*   **Zephyr**: A polished, smooth academic narrative with clean transitions and descriptive observations.
